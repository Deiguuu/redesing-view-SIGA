Registro de Calificaciones

Seleccionar Curso y Actividad

Instrucciones

  1. Seleccione el curso para el cual desea registrar calificaciones.
  2. Seleccione la actividad específica que está calificando.
  3. Ingrese las calificaciones para cada estudiante (valores entre 0 y 10).
  4. Puede guardar calificaciones individualmente o todas a la vez.
  5. Las calificaciones guardadas se marcarán con un indicador verde.
  6. Puede modificar calificaciones ya guardadas si es necesario.
Consejo para ahorrar tiempo:
Utilice la tecla Tab para navegar rápidamente entre los campos de calificación.
